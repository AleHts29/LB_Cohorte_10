## Ejercicio salas


Test 1: Agregar productos iniciales – Se agregan productos al inventario y se verifica que las cantidades coincidan con las esperadas.

Test 2: Verificar stock suficiente – Se comprueba si hay suficiente stock para ciertos productos, asegurándose de que la verificación funcione correctamente.

Test 3: Retiro de stock insuficiente – Se verifica que el inventario indique insuficiencia cuando se intenta retirar más de lo disponible.

Test 4: Retirar productos y verificar estado final – Se retiran productos del inventario y se valida que las cantidades finales sean correctas.
